## 07 浮点类型

浮点类型：

|  浮点类型   | 宽度 | 字节 |
| :---------: | :--: | :--: |
|    float    | 32位 |  4   |
|   double    | 64位 |  8   |
| long double | 64位 |  8   |

```c
float f = 1.23F;
double d = 4.56;
long double l = 7.89L;
```

IEEE编码规范

float：32位

|   31   | 30 ~ 23  |  22 ~ 0  |
| :----: | :------: | :------: |
|  1位   |   8位    |   23位   |
| 符号位 | 指数部分 | 尾数部分 |

double：64位

|   63   | 62 ~ 52  |  51 ~ 0  |
| :----: | :------: | :------: |
|  1位   |   11位   |   52位   |
| 符号位 | 指数部分 | 尾数部分 |

转浮点类型：
	整数部分：一定可以完整地转换成二进制
	小数部分：不可能做到完全精确

8.25转float类型：
	整数部分：1000（8）
	小数部分：0.01（0.25）
	1000.01 → 1.00001 * 2<sup>3</sup>
		符号位：0
		指数部分：10000010
		小数部分：00001000000000000000000
		→ 0  10000010  00001000000000000000000
		→ 0100  0001  0000  0100  0000  0000  0000  0000
		→ 0x41040000
<img src="images/image-20240131015126901.png" style="zoom:50%;" />

精度：尾数的位数
	float：7位有效数字，2<sup>23</sup> = 8388608（7位）
	double：16位有效数字，2<sup>52</sup> = 4503599627370496（16位）

## 13 switch语句为什么高效

条件少：switch语句与if-else语句几乎没有区别
条件多：switch语句比if-else语句高效

```c
void main() {
	int x = 1;
	int y = 2;
	if (x == 7) {
		y += 1;
	}
	else if (x == 34) {
		y += 2;
	}
	else if (x == 27) {
		y += 3;
	}
	else if (x == 48) {
		y += 4;
	}
	else {
		y += 5;
	}
	return;
}
```
